# Reports — setbrain Cell

Point-in-time analysis that informs decisions. Not chronological records
(`history/`), not curated patterns (`library/`), not completion evidence
(`proofs/`).

Cell-wide analysis lives here. Analysis about setbrain content specifically
lives in `apps/setbrain/reports/`.

This example Cell has no cell-wide report yet — the surface exists because the
Cell is agent-operated and analysis will accumulate. When a report produces a
reusable insight, distill the insight to `library/`; the report stays here as
source evidence.
