# Pattern — One-Page Set Brief

*(Fictional example. Distilled from TASK-20260701-001.)*

A set brief that promoters accept without follow-up questions has exactly
five parts, in this order:

1. **Hook** — one sentence: the feel of the set, no genre jargon.
2. **Genre line** — primary genre + up to two sub-styles, from
   `apps/djbrain/knowledge/genre-notes.md` vocabulary.
3. **Arc** — opening energy → peak → closing, three short phrases.
4. **Venue fit** — one sentence tying the arc to the venue profile.
5. **Practicals** — duration, BPM range, equipment needs. Text only — no
   embedded images (HUMAN-GATE-20260630-001).

This entry is a *distilled copy*. The operating original — the prompt that
generates briefs — stays in `apps/djbrain/prompts/set-brief.prompt.md`.
Library entries never contain the live capability; they teach the pattern.
